## 下载功能

目前下载功能是演示模式。在实际部署时，需要：

1. 将游戏文件上传到服务器或 CDN
2. 更新 `startDownload()` 函数中的 URL
3. 实现下载统计和日志记录
4. 添加镜像服务器以提高下载速度

## 使用方式

直接在浏览器中打开 `index.html` 文件即可查看网站。

## 自定义

### 修改下载链接
在 JavaScript 中的 `startDownload()` 函数中修改 URLs 对象。

### 修改游戏信息
直接编辑 HTML 中的游戏数据，包括：
- 游戏标题和描述
- 角色信息
- 系统要求
- 评论内容

### 添加实际图片
替换占位符区域为真实的游戏截图和海报。
