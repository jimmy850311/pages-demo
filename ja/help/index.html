ダウンロードとインストール

ステップ1： ダウンロードページから適切なバージョン（Windows/Mac/Linux/Android）を選択します。

ステップ2： ファイルをダウンロードして、お好みのフォルダに保存します。

ステップ3： ZIPファイルを解凍します。

ステップ4： 実行ファイル（.exe / .app / 実行スクリプト）をダブルクリックして起動します。

インストール画面が表示されない場合、ファイルが正しく解凍されているかご確認ください。

⚙️

システム要件

最小要件：
- OS: Windows 10以降 / macOS 10.10以降
- CPU: 2コア以上 2GHz以上
- メモリ: 8GB以上
- ストレージ: 5GB空き容量

推奨要件：
- OS: Windows 11以降 / macOS 10.12以降
- CPU: 4コア以上 3GHz以上
- メモリ: 16GB以上
- ストレージ: 8GB空き容量（SSD推奨）

🔐

セキュリティとウイルス

Helping the Hotties は安全なゲームです。ダウンロードやインストール時にウイルス警告が表示される場合、これはセキュリティソフトの誤検知の可能性があります。

公式配布ページからのダウンロードをお勧めします。疑問がある場合は、お問い合わせフォームからご連絡ください。

🎮

ゲームがすぐに終了する

このような場合は以下をお試しください：

1. 別のフォルダにゲームを再度解凍します
2. グラフィックドライバを最新に更新します
3. その他のアプリケーションをすべて閉じてからゲームを起動します
4. 管理者権限で実行を試みます

それでも解決しない場合、お問い合わせフォームから詳細をお知らせください。

🔊

音声が出ない

音声が再生されない場合：

1. ゲーム内の音量設定を確認します
2. PC/Mac本体のボリュームが十分か確認します
3. オーディオドライバが最新か確認します
4. ゲームを再起動してみます

オーディオ形式の互換性の問題の場合があります。詳細はお問い合わせください。

📱

Android版のトラブル

インストールできない場合：
- 提供元不明のアプリの許可設定を確認します
- 十分なストレージ容量があるか確認します
- キャッシュを削除してから再度インストールを試みます

起動しない場合：
- アプリデータを削除して再度起動します
- Androidを最新版に更新します

💾

セーブファイルの管理

セーブファイルはゲームフォルダ内の「saves」フォルダに保存されます。

複数のデバイスでプレイする場合は、セーブファイルをコピーして他のデバイスにも配置できます。ただし、同一セーブでの複数デバイス同時使用はお避けください。

🌐

言語の変更方法

ゲーム内で言語を切り替えられます：

1. メインメニューから「設定」を選択
2. 「言語」オプションを探す
3. お好みの言語を選択
4. ゲームを再起動

日本語、英語、中国語、スペイン語に対応しています。

🆘

さらにサポートが必要な場合

このページで解決しない問題がある場合は、お問い合わせフォームからご連絡ください。

以下の情報を含めていただくと、スムーズなサポートが可能です：
- ご使用のOS（Windows/Mac/Linux等）
- ゲームのバージョン
